***********Project name: Servis Racunara (Computer Service)***********


***********Developers: Aleksa Marceta and Aleksandar Panzalovic***********

***********Short Description: This project resulted as a task set by our professor as exam requirement. It's main purpose is to print bills of landing for a Computer Service company.  You can view all issued bills of landing and save pdf if customer has lost his. There is also section for viewing and adding new goods. Section that is also present but not planned and not finished is Administrative zone. This zone shall include adding new members (workers), viewing all currently employed and fixing any password related issues by local administrator. ***********
